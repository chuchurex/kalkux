# Kilig Coffee - Ejemplo Kalkux

Primer cliente de Kalkux. Tostadores de cafe de especialidad en Santiago de Chile.

- **Repo**: https://github.com/chuchurex/kilig
- **Store ID**: `kilig`
- **Deploy**: kilig.chuchurex.cl (Cloudflare Pages)
- **Moneda**: CLP
- **Productos**: 18 (6 cafes con variantes, 12 accesorios)
- **Variantes**: peso (4 opciones), molienda (9 opciones)
- **Pago**: Snipcart + MercadoPago
